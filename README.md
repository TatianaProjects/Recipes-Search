# 🍳 Recipe Search

A modern React app that helps users search for recipes using the Edamam API.  
Find meal ideas, ingredients, and cooking inspiration in a clean and responsive interface.

---

## ✨ Features

- Recipe search by keyword
- API integration with `fetch()`
- React Hooks (`useState`, `useEffect`)
- Dynamic recipe cards
- Responsive design
- Video background UI
- Interactive search functionality

---

## 🛠 Technologies

- React
- JavaScript
- CSS
- Edamam API
- React Icons
